# Simple-SpringTodoListApi
간단한 To-Do-List

-------------------------------

### 프로젝트 소개
스프링 부트를 이용해 간단한 Todo List backend api를 구축하였습니다. 


(프론트엔드는 Todo-backend 사이트를 이용하였기 때문에 작성하지 않았습니다.)


[Todo-backend 사이트](https://todobackend.com/)

***사용 기술 스택***|
---|
java|
springboot|
spring data jpa|
Todo-backend|
